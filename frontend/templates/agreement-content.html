Пользовательское соглашение

Общие положения

1.1. Настоящее Пользовательское соглашение (далее — «Соглашение») регулирует отношения между пользователем (далее — «Пользователь») и сервисом «» (далее — «Сервис») по использованию мини-приложения для генерации видео из изображений.

1.2. Используя Сервис, Пользователь подтверждает своё согласие с условиями настоящего Соглашения.

2. Условия использования

2.1. Сервис предназначен для создания видео из загружаемых Пользователем изображений с использованием технологий искусственного интеллекта.

2.2. Пользователь гарантирует, что обладает всеми необходимыми правами на загружаемые изображения.

2.3. Запрещено использование Сервиса для:

  • Создания контента, нарушающего законодательство РФ
  • Генерации материалов, содержащих насилие, дискриминацию или порнографию
  • Нарушения авторских прав третьих лиц
  • Мошеннических действий

Права и обязанности

Сервис обязуется:
  • Обеспечивать работоспособность приложения
  • Защищать персональные данные Пользователя
  • Предоставлять техническую поддержку
Пользователь обязуется:
  • Использовать Сервис по назначению
  • Не предпринимать попыток взлома
  • Не загружать вредоносное ПО

Платежи и тарифы

4.1. Сервис предоставляет услуги на платной основе согласно действующим тарифам.

4.2. Все платежи производятся через платёжную систему ВКонтакте.

4.3. Средства за невыполненные задания возвращаются на баланс Пользователя.

4.4. Сервис оставляет за собой право изменять тарифы с уведомлением Пользователей.

Конфиденциальность

5.1. Сервис обрабатывает персональные данные в соответствии с Политикой конфиденциальности.

5.2. Загружаемые изображения хранятся на серверах ограниченное время, необходимое для обработки.

5.3. Сервис не передаёт персональные данные третьим лицам без согласия Пользователя, кроме случаев, предусмотренных законодательством.

Авторские права

6.1. Исходные изображения остаются собственностью Пользователя.

6.2. Созданные видео становятся собственностью Пользователя при условии соблюдения авторских прав на исходные материалы.

6.3. Сервис сохраняет право использовать созданные видео в анонимизированном виде для улучшения алгоритмов.

Ограничение ответственности

Сервис не гарантирует:
  • Бесперебойную работу приложения
  • Соответствие результатов ожиданиям
  • Сохранность данных при форс-мажоре
Сервис не несёт ответственности за:
  • Содержание загружаемых изображений
  • Нарушение авторских прав
  • Потерю данных по вине Пользователя

9.1. Сервис оставляет за собой право изменять условия Соглашения.

9.2. Изменения вступают в силу с момента публикации новой версии Соглашения.

9.3. Продолжение использования Сервиса после изменений означает согласие с новыми условиями.

Заключительные положения

10.1. Соглашение действует с момента начала использования Сервиса.

10.2. Все споры решаются путём переговоров.

10.3. При неурегулировании споров они передаются на рассмотрение в суд по месту нахождения Сервиса.

10.4. Сервис вправе приостановить или прекратить доступ Пользователя при нарушении условий Соглашения.
